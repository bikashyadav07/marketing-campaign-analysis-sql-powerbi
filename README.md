# Marketing Campaign Analysis using SQL & Power BI

## Project Overview
This project analyzes marketing campaign performance using SQL and Power BI to generate business insights and interactive dashboards.

## Tools Used
- SQL
- Power BI
- Excel/CSV Dataset

## Key Features
- Data Cleaning
- KPI Analysis
- CTR Analysis
- Category-wise Performance
- Monthly Trend Analysis
- Interactive Dashboard

## SQL Concepts Used
- GROUP BY
- Aggregate Functions
- Window Functions
- Ranking
- CTEs

## Dashboard Insights
- Top Performing Categories
- Engagement Trends
- Campaign Performance
- Click Through Rate Analysis
